# 0xRemyRuiz.github.io

Personal core dump, used to illustrate my learning process and personal projects, and maybe more — [0xremyruiz.github.io](https://0xremyruiz.github.io)

## Stack

- [Hugo](https://gohugo.io/) (extended) + [PaperMod](https://github.com/adityatelange/hugo-PaperMod)
- Deployed on GitHub Pages via GitHub Actions

## Quick start

```bash
git clone --recurse-submodules https://github.com/0xRemyRuiz/0xRemyRuiz.github.io
hugo server -D
```

If cloned was done without `--recurse-submodules` :

```bash
git submodule update --init --recursive
```
